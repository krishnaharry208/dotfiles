# Krishnaharry Dotfiles

A collection of personal configuration files and an automated installation script to set up a Linux/Unix environment quickly.

## 🚀 Quick Start

To install these dotfiles on a new system, run the following command in your terminal:

```bash
git clone [https://github.com/krishnaharry208/dotfiles](https://github.com/krishnaharry208/dotfiles)
cd dotfiles
sudo ./install.sh



#if you want to install one command run this command
curl -fsSL https://raw.githubusercontent.com/krishnaharry208/dotfiles/main/install.sh | sudo bash
